# [![PAC_Hibernate](https://github.com/Zafion/PAC_Hibernate/blob/main/recursos/hibernate_64px.png)](https://github.com/Zafion/PAC_Hibernate) Aplicación de Gestión de Bases de Datos usando Hibernate

Desarrollada con Java utilizando el framework Hibernate y el IDE Eclipse.

Trabajo de desarrollo realizado durante el segundo semestre del Ciclo Superior de Desarrollo de Aplicaciones Multiplataforma.
Corresponde a la asignatura de Acceso a Datos.

La práctica consiste en crear una aplicación de gestión de bases de datos. A través del framework Hibernate, se deben crear unas tablas específicas en la base de datos. Una vez creadas, se deben introducir los datos indicados en cada una de las tablas e informar de su realización.

Las instrucciones detalladas del ejercicio se pueden consultar en el archivo 
https://github.com/Zafion/PAC_Hibernate/blob/main/1s2223_M06_PAC%20DESARROLLO.pdf

</br>
